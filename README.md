### Descripción General
*Desarrollo de una solución integral en Salesforce para la gestión y comercialización de tractocamiones seminuevos. El sistema centraliza el inventario y ofrece una experiencia de usuario dinámica mediante un catálogo interactivo que facilita la búsqueda y localización geográfica de unidades en tiempo real.*

### Componentes Clave
**Catálogo Digital Avanzado:** Implementación de una interfaz visual (LWC) para mostrar el inventario, optimizada con paginación eficiente y filtros avanzados (marca, modelo, kilometraje, año).

**Geolocalización Inteligente:** Integración de un Mapa Global (utilizando el componente lightning-map) que consume las coordenadas de cada registro para visualizar la distribución física de la flota.

**Arquitectura de Datos:** Configuración de objetos personalizados con campos de geolocalización, gestión de estados de venta y automatización de seguimiento de leads.
